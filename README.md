# Базы данных — итоговый проект

## Описание проекта
Проект содержит решения 13 задач по работе с 4 базами данных:
1. **Транспортные средства** (автомобили, мотоциклы, велосипеды)
2. **Автомобильные гонки**
3. **Бронирование отелей**
4. **Структура организации**

Каждая база данных включает создание таблиц, наполнение тестовыми данными и SQL-запросы к ним.

## Структура базы данных

### База 1: Транспортные средства
- **Vehicle** — производители и модели
- **Car** — автомобили (vin, объём, мощность, цена, трансмиссия)
- **Motorcycle** — мотоциклы (vin, объём, мощность, цена, тип)
- **Bicycle** — велосипеды (серийный номер, количество передач, цена, тип)

### База 2: Автомобильные гонки
- **Classes** — классы автомобилей (тип, страна, количество дверей, объём, вес)
- **Cars** — автомобили (название, класс, год)
- **Races** — гонки (название, дата)
- **Results** — результаты (автомобиль, гонка, позиция)

### База 3: Бронирование отелей
- **Hotel** — отели (название, локация)
- **Room** — номера (тип, цена, вместимость)
- **Customer** — клиенты (имя, email, телефон)
- **Booking** — бронирования (даты заезда/выезда)

### База 4: Структура организации
- **Departments** — отделы
- **Roles** — роли сотрудников
- **Employees** — сотрудники (иерархия через ManagerID)
- **Projects** — проекты
- **Tasks** — задачи

## Функциональность

### База 1 — Транспортные средства
- Задача 1: Найти спортивные мотоциклы мощнее 150 л.с. дешевле 20000$
- Задача 2: Объединить данные по автомобилям, мотоциклам и велосипедам с разными критериями

### База 2 — Автомобильные гонки
- Задача 1: Для каждого класса автомобиль с лучшей средней позицией в гонках
- Задача 2: Абсолютный лидер гонок по средней позиции
- Задача 3: Классы с минимальной средней позицией
- Задача 4: Автомобили с позицией лучше средней по классу (минимум 2 авто в классе)
- Задача 5: Классы с наибольшим количеством автомобилей с позицией > 3.0

### База 3 — Бронирование отелей
- Задача 1: Клиенты с более чем 2 бронированиями в разных отелях
- Задача 2: Клиенты, потратившие более 500$ и имеющие >2 бронирований в разных отелях
- Задача 3: Классификация клиентов по предпочтениям (дорогие/средние/дешёвые отели)

### База 4 — Структура организации
- Задача 1: Рекурсивный вывод всех подчинённых Ивана Иванова
- Задача 2: То же + количество задач и прямых подчинённых
- Задача 3: Менеджеры с подчинёнными (рекурсивный подсчёт всех подчинённых)
